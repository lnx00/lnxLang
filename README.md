# lnxLang
A simple programming language written in C#

## Features
- Global and local variables (string, int, float, bool)
- if-Statements
- while-loops
- functions

## Example
```
global a -> int = 5;

func $main() -> void {
  PrintLine("a is " + a);
  local b -> int = 0;
  b = a + 10;
  PrintLine("b is " + b);
}
```
